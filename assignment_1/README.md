### Pre-requisite
- python should be installed
- Preferred IDE should be present for better readability and execution (example: VSCode)
- Set virtual environment (.venv) before execution
- Download the .zip file and extract all the files to a specific location (say: C:/users/ai)
- Open IDE or Terminal on that location and follow the further instructions

### How to install dependencies ?
 All required libraries are defined in `requirement.txt` file. Run the below command on Terminal
>  use python / python3 according to the python version installed

-  python3 -m pip install -r requirement.txt   

### How to Run?
> Command to run file individually
- For `part1_zipf.py` : 
    > python ./source/part1_zipf.py

- For `part2_language_model.py` : 
    > python ./source/part2_language_model.py

- For `part3_hmm_tagger.py` : 
    > python ./source/part3_hmm_tagger.py

#### To run using runner.py
    - python runner.py
    - Enter the file name you wish to run (ex: part1_zipf)

### Output

The assignment's output will store in a folder called `output`, which will be automatically created (if not already) upon running `part1_zipf.py`
